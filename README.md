# Drone Pilot Application

An Android-based application designed to assist drone pilots in managing agricultural spraying operations efficiently. The app integrates Firebase services and provides features for user authentication and data handling.

---

##  Features

- User Signup & Login (Firebase Authentication)
- Firestore Integration (data storage)
- Drone operation data handling
- MAVLink communication with drone
- Telementry monitoring
- SITL-based drone simulation testing

---

## Tech Stack

- **Language:** Kotlin
- **IDE:** Android Studio
- **Backend:** Firebase (Authentication + Firestore)
- **Build System:** Gradle
- **Protocol:** MAVLink

---



### Prerequisites
- Android Studio installed
- Firebase project setup
